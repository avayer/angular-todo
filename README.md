# AngularTodo

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.1.

This is a simple Todo-List.

## Pre-requisites

-NodeJs

## How to run

- git clone https://github.com/avayer/angular-todo.gitrun
- run npm install 
- run ng serve
